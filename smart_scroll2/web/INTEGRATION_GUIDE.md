## Интеграция в существующую систему

### Навигация
Новая секция "Документы" добавлена в:
- Единую навигационную панель (`unified-navbar.html`)
- Нижнюю навигацию (`navbar.html`)

### Стили
Страница использует собственные стили, встроенные в HTML для:
- Адаптивности

## Тестирование

### Проверка навигации
1. Откройте любую страницу с навигацией
2. Убедитесь, что появилась ссылка "Документы"
3. Проверьте переход на `document-reader.html`

### Тестирование функциональности
1. Откройте `document-reader.html`
2. Введите тестовый OpenAI API ключ
3. Загрузите тестовый PDF или Word документ
4. Задайте тестовый вопрос

### Тестовые файлы
Для тестирования можно использовать:
- Любой PDF документ
- Любой Word документ (.docx, .doc)
- Текстовые файлы (не поддерживаются)

## Возможные проблемы

### CORS ошибки
- Убедитесь, что страница открывается через HTTP/HTTPS сервер
- Не открывайте файл напрямую в браузере

### API ошибки
- Проверьте правильность OpenAI API ключа
- Убедитесь в наличии средств на балансе

### Библиотеки
- PDF.js и Mammoth.js загружаются с CDN
- Требуется интернет-соединение
