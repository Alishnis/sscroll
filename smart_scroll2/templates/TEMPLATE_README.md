# Smart Scroll - HTML Шаблон

Этот шаблон предназначен для создания новых страниц в приложении Smart Scroll.

## Структура шаблона

### Основные компоненты:
- **Header** - заголовок страницы с логотипом
- **Main Content** - основная область контента
- **Right Sidebar** - боковая панель с иконками
- **Sidebar Content** - карточки контента (Содержание, Заметки, Квиз)
- **Navigation** - нижняя навигационная панель

## Как использовать шаблон

### 1. Создание новой страницы
```bash
cp template.html your-page-name.html
```

### 2. Обновление метаданных
Замените следующие элементы в `<head>`:
```html
<title>Smart Scroll - [Название страницы]</title>
```

### 3. Обновление заголовка
Замените в header:
```html
<h1 class="header__title">[Название страницы]</h1>
```

### 4. Обновление основного контента
Замените в `.content-card`:
```html
<h2 class="content-card__title">[Заголовок контента]</h2>
<p class="content-card__text">[Основной текст контента страницы...]</p>
```

### 5. Обновление боковых панелей
- **Содержание**: обновите текст в `.summary-card__text`
- **Заметки**: placeholder уже настроен
- **Квиз**: обновите вопрос и варианты ответов

### 6. Настройка навигации
Измените активный элемент в navbar:
```html
<div class="navbar__item navbar__item--active">
```

## Стили

Все стили уже подключены через `styles.css`. Дополнительные стили для новых элементов:

### Основная карточка контента
```css

    /* Основная карточка с контентом */
}

.content-card__header {
    /* Заголовок карточки */
}

.content-card__title {
    /* Заголовок контента */
}

.content-card__body {
    /* Тело карточки */
}

.content-card__text {
    /* Основной текст */
}
```
