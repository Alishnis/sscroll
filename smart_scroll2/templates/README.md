# Smart Scroll - Лента рекомендаций

Веб-приложение для просмотра рекомендованного видео контента с современным интерфейсом.

## Структура проекта

```
├── index.html          # Главная HTML страница
├── styles.css          # Стили с BEM методологией
├── script.js           # JavaScript для интерактивности
├── images/             # Папка с изображениями
│   └── video-thumbnail.jpg
└── README.md          # Документация
```

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - стили с BEM naming convention
- **Vanilla JavaScript** - интерактивность без фреймворков

## BEM Naming

Проект использует методологию BEM (Block Element Modifier):

### Блоки:
- `page` - основной контейнер страницы
- `header` - шапка сайта
- `main` - основной контент
- `video-card` - карточка видео
- `sidebar` - боковая панель
- `navbar` - навигационная панель

### Элементы:
- `header__title` - заголовок в шапке
- `header__logo` - логотип
- `video-card__thumbnail` - миниатюра видео
- `video-card__title` - название видео
- `video-card__action` - кнопка действия
- И т.д.

### Модификаторы:
- `navbar__item--active` - активный элемент навигации
- `video-card__action--likes` - кнопка лайков
- `video-card__action--share` - кнопка поделиться

## Функциональность

### Интерактивные элементы:

1. **Навигация** - переключение между разделами
2. **Боковая панель** - выбор категорий контента
3. **Действия с видео**:
   - Лайки
   - Просмотры
   - Содержание
   - Источник
   - Квиз
   - Поделиться

### Анимации:

- Плавное появление элементов при скролле
- Анимация при клике на кнопки
- Hover эффекты на интерактивных элементах
- Уведомления с анимацией появления/исчезания

## Адаптивность

Сайт полностью адаптивен для разных размеров экранов:

- **Desktop** (1920px+) - полная версия
- **Laptop** (1440px) - уменьшенные отступы
- **Tablet** (1024px) - вертикальное расположение контента
- **Mobile** (768px и меньше) - мобильная версия

## Запуск

Просто откройте файл `index.html` в любом современном браузере.

Или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Затем откройте http://localhost:8000 в браузере.
